# M3U8视频下载器

一个基于PyQt6开发的M3U8视频下载工具，支持多线程下载、断点续传、自动合并等功能。

![应用截图](screenshots/main.png)

## 功能特点

- 🚀 多线程并发下载，显著提升下载速度
- 💾 支持断点续传，意外中断可继续下载
- 🔄 自动合并下载片段为MP4文件
- 📝 自定义请求头，支持各类网站
- 🎨 现代化界面设计，支持暗黑模式
- 📊 实时下载进度显示
- 📋 详细的日志记录
- ⏯️ 支持暂停/继续下载
- 🔧 可配置下载线程数

## 系统要求

- macOS 10.13 或更高版本
- Python 3.8 或更高版本
- FFmpeg（用于视频合并）
